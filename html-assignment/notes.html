HTML Tags

Paragraph Tag
We use Paragraph Tag as the term very well says to create a paragraph of text and you can create as many as you want. The browser will add a space before and after the content of the paragraph.
Superscript Text
We use this tag to define a text as superscript. Example: 13th
Heading
Are headings tags that define the hierarchy of the content. We can use the h1 tag for the most important heading of the page and this is aldo the Title. For the next heading level or subtitles, we have h2, h3, h4, h5 and h6. The browser will style each heading differently, stating with the biggest size for h1 then h2 and so on.
Abbreviation
This tag is used to shorten terminologies or words. Example: Suppose we want to shorten my name which is Daniela by my nickname Dani, so, this tag understands Daniela as Dani.
Blockquote
This is a large, stand-alone quote from another source.
